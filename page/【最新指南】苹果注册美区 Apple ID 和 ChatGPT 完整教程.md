### 1. 创建 Apple ID

1. 使用 Safari 浏览器访问 Apple ID 注册页面
2. 填写基本信息（姓名、邮箱、密码等）
3. 完成手机号和邮箱验证

1. 登录新创建的 Apple ID
2. 进入"个人信息"设置
3. 选择"国家或地区"
4. 将地区更改为"美国"
